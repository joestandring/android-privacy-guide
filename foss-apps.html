Bromite

Bromite is a browser based on Chromim which patches out Google features that do not respect privacy (similar to ungoogled-chromium). It also features additional privacy improvements such as a built in ad/tracker blocker using filters from multiple sources, DNS-over-HTTPS support, and other privacy patches. As it is built on Chromium, it performs better than Firefox-based browsers in the majority of situations and does not require patches to be implemented manually via user.js, add-ons, or about:config. Because of this however, it can be harder to customise various settings and features manually compared to Firefox.
